Poptracker for Funi Raccoon Game.

Mod can be installed [here](https://github.com/Jeffdev-Archipelago-Implementations/FuniRaccoonAP/releases/latest)

APWorld can be installed [here](https://github.com/Jeffdev-Archipelago-Implementations/Archipelago-Funi/releases/latest)

Only compatible with Mod and APWorld v1.4.0 or higher. Will not function properly with any older version.
